# SyntaxWear — Landing Page de E-commerce

## Descrição

Projeto de front-end estático para uma landing page de e-commerce chamada SyntaxWear. A página é construída com HTML e CSS, com layout responsivo, navegação fixa, seção de destaque, cards de produto, categorias e rodapé.

## Estrutura do projeto

- `index.html` — página principal
- `css/reset.css` — reset básico de estilos
- `css/variables.css` — variáveis de CSS e fontes
- `css/base.css` — estilos globais e utilitários
- `css/components/header.css` — estilos do cabeçalho e menu responsivo
- `css/components/hero.css` — estilos da seção principal (hero)
- `css/components/product-card.css` — estilos dos cards de produto
- `css/components/product-grid.css` — estilos do grid de produtos
- `css/components/panel.css` — estilos de painel adicional (importado no HTML)
- `css/components/newsletter.css` — estilos do formulário de inscrição
- `css/components/footer.css` — estilos do rodapé
- `images/` — imagens usadas no layout (banners, ícones, logo e produtos)

## Funcionalidades

- Layout responsivo para desktop e mobile
- Navegação fixa com menu mobile acionado por checkbox
- Seção hero com imagem de destaque e CTA
- Cards de categorias com imagens de fundo
- Grid de produtos em CSS Grid
- Rodapé com newsletter e links de navegação

## Como visualizar

1. Abra o arquivo `index.html` diretamente no navegador.
2. Ou use uma extensão de servidor local como Live Server no VS Code.

## Observações

- Não há scripts JavaScript neste projeto.
- O CSS é organizado em componentes para facilitar manutenção.
- As imagens estão em `images/` e referenciadas pelos estilos e pelo HTML.

## Créditos

Criado como projeto de estudo/desenvolvimento front-end para uma página de e-commerce.
